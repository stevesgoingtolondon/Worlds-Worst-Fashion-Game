# Worlds-Worst-Fashion-Game
Overview
--------------------------------------
Did you ever play thoses stupid flash games as a kid? Well, this is a parody of them. User will be given a character and clothing to make them an outfit for random prompt. The prompts will be complete nonsense, and really this is just a time waster game. I want to try and incorporating HTML Canvas to add animations and focus more on graphics.

Features
--------------------------------------
- Creates a random selection of 10 items each game
- 20+ random promts
- Unique graphics and animations
- It'll be funny probably
- Hand drawn sprites
